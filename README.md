LiterAlura
Descripción:
El proyecto LiterAlura es una aplicación para gestionar y consultar información sobre libros y autores. 
Ofrece un menú interactivo con diversas opciones para buscar libros por título, listar libros y autores 
registrados, consultar autores vivos en un determinado año, listar libros por idioma, y ver el top 5 
de libros más descargados.

Funcionalidades:

Buscar libro por título: Permite buscar y registrar libros a partir de un título proporcionado por el usuario.
Listar libros registrados: Muestra un listado de todos los libros almacenados en la base de datos.
Listar autores registrados: Muestra un listado de todos los autores registrados en la base de datos.
Listar autores vivos en un determinado año: Permite consultar qué autores estaban vivos en un año específico.
Listar libros por idioma: Muestra los libros registrados filtrados por el idioma seleccionado por el usuario.
Listado de libros por autor: Muestra los libros escritos por un autor específico seleccionado por el usuario.
Top 5 libros más descargados: Presenta un listado de los cinco libros más descargados.
